# Saphir Backend
Ce projet consiste en la création d'une base de données pour ensuite insérer les données de deux types de sources (Météo France et Barani).

## commener 
### Prerequisites
avans de commencer assurer d'avoir 

- Python 3.10.11
- flask 
- PostgreSQL 15
